## 功能描述

当访问基金详情页时，如果该基金数据不存在，系统自动触发采集流程：
1. Java 检测到基金不存在
2. 调用 Python 采集服务
3. Python 采集基础信息 + 净值历史 + 计算指标
4. 返回采集后的基金数据
